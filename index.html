⚠️ 為什麼別人填的資料看不到？

本系統使用瀏覽器的 localStorage 存放本機資料，A 電腦填的資料只在 A 電腦看得到。要讓管理員看到所有人的申請，必須設定 Google Sheets 作為共用後端。設定完成後，每次提交都會即時寫入 Sheets，管理員點「從雲端同步」即可看到所有人的資料。

步驟一：建立 Google 試算表

1. 前往 Google Sheets，新增空白試算表
2. 將第一個工作表命名：申請記錄
3. A1 開始填入標題：申請ID | 單位 | 姓名 | 裝備 | 規格 | 損壞描述 | 汰換原因 | 狀態 | 管理員備註 | 申請時間 | 審核時間
4. 新增第二個工作表命名：庫存
5. A1 填入：規格ID | 裝備名稱 | 規格 | 庫存數量 | 最後更新

步驟二：貼上 Apps Script

在試算表點「擴充功能」→「Apps Script」，貼上以下程式碼：
const SS_ID = SpreadsheetApp.getActive().getId(); function doPost(e) { const data = JSON.parse(e.postData.contents); const ss = SpreadsheetApp.openById(SS_ID); if (data.action === 'addRequest') { ss.getSheetByName('申請記錄').appendRow([ data.id, data.unit, data.name, data.equipName, data.specs, data.condition, data.reason, '待審核', '', data.submittedAt, '' ]); } if (data.action === 'updateRequest') { const sh = ss.getSheetByName('申請記錄'); const rows = sh.getDataRange().getValues(); for (let i = 1; i < rows.length; i++) { if (rows[i][0] == data.id) { sh.getRange(i+1,8).setValue(data.status==='approved'?'已核准':'未核准'); sh.getRange(i+1,9).setValue(data.note||''); sh.getRange(i+1,11).setValue(new Date().toLocaleString('zh-TW')); break; } } } if (data.action === 'updateStock') { const sh = ss.getSheetByName('庫存'); const rows = sh.getDataRange().getValues(); let found = false; for (let i = 1; i < rows.length; i++) { if (rows[i][0] == data.specId) { sh.getRange(i+1,4).setValue(data.stock); sh.getRange(i+1,5).setValue(new Date().toLocaleString('zh-TW')); found = true; break; } } if (!found) { sh.appendRow([data.specId, data.equipName, data.spec, data.stock, new Date().toLocaleString('zh-TW')]); } } return ContentService.createTextOutput(JSON.stringify({ok:true})).setMimeType(ContentService.MimeType.JSON); } function doGet(e) { const ss = SpreadsheetApp.openById(SS_ID); const action = e.parameter.action; if (action === 'ping') return out({ok:true}); if (action === 'getAll') { const reqRows = ss.getSheetByName('申請記錄').getDataRange().getValues().slice(1); const invRows = ss.getSheetByName('庫存').getDataRange().getValues().slice(1); const toStatus = s => s==='已核准'?'approved':s==='未核准'?'rejected':'pending'; return out({ requests: reqRows.filter(r=>r[0]).map(r=>({ id:String(r[0]),unit:r[1],name:r[2],equipName:r[3], specs:r[4],condition:r[5],reason:r[6], status:toStatus(r[7]),adminNote:r[8]||'',submittedAt:r[9] })), stockRows: invRows.filter(r=>r[0]).map(r=>({ specId:String(r[0]),equipName:r[1],spec:r[2],stock:Number(r[3]),updated:r[4] })) }); } return out({ok:false}); } function out(obj){ return ContentService.createTextOutput(JSON.stringify(obj)).setMimeType(ContentService.MimeType.JSON); }

步驟三：部署

1. 點「部署」→「新增部署作業」→ 類型選「網頁應用程式」
2. 執行身份：我　存取權：所有人
3. 點「部署」→ 複製網址 → 貼到上方設定欄 → 儲存
⚠️ 修改 Script 後必須重新部署（管理現有部署 → 編輯 → 版本選「新版本」）才會生效
